Given a string ‘str’ of digits and an integer ‘n’, build the lowest possible number by removing ‘n’ 
digits from the string and not changing the order of input digits.

Examples:

Input: str = "4325043", n = 3
Output: "2043"

Input: str = "765028321", n = 5
Output: "0221"

Input: str = "121198", n = 2
Output: "1118"
